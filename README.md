# Smart Office Booking System

## 📌 Project Description
This project is developed using Java, HTML, CSS and MySQL.
It allows users to book office seats online.

## 🚀 Features
- User Login
- Seat Booking
- Admin Panel
- OTP Verification
- Database Integration

## 🛠️ Technologies Used
- Java
- HTML
- CSS
- MySQL
